# HOW TO ADD A NEW SOURCE

1. Find the appropriate section (government → by country, industry → by sector, news → by language)
2. Add a row with: name, URL, type, language, frequency, notes
3. If it has RSS, add the feed URL to the RSS section
4. If it produced real events, consider promoting it to a higher tier
5. If it stopped producing after 3 checks, add a note but keep it for periodic re-checks
6. For non-English sources, always note the language
7. For government sources, note if they have an API or downloadable data
